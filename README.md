# Marginalia

Your reading-notes companion built with Vite + React + TypeScript + Tailwind/shadcn-ui.

## Local development

```sh
# 1) Install dependencies
npm ci

# 2) Start the dev server (http://localhost:8080)
npm run dev
```

## Build and preview

```sh
npm run build
npm run preview
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
